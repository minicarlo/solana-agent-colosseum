## Recording Instructions

### Option 1: Screen Recording (Recommended)
1. Open https://minicarlo.github.io/zephyr/ in browser
2. Use OBS, Loom, or QuickTime to record screen
3. Record at 1080p, 30fps
4. Add voiceover using the script above
5. Export as MP4 (H.264 codec)

### Option 2: Animated Video
1. Use tools like:
   - Canva (free video templates)
   - Loom (browser-based recording)
   - CapCut (mobile/desktop editing)
   - VEED.io (online editor)
2. Import screenshots from the website
3. Add text overlays with the script
4. Add background music (royalty-free)
5. Export as MP4

### Technical Specs
- **Duration:** 45-60 seconds
- **Resolution:** 1920x1080 (1080p)
- **Format:** MP4 (H.264)
- **Audio:** Clear voiceover + optional background music
- **File Size:** Under 100MB (for easy uploading)

## Assets Available
- Landing page: https://minicarlo.github.io/zephyr/
- GitHub repo: https://github.com/minicarlo/zephyr
- Logo/SVG assets in repo

## Quick Recording Tips
1. **Voiceover:** Speak clearly, moderate pace
2. **Music:** Use low volume background track (20-30% volume)
3. **Text:** Keep on screen for at least 3 seconds per text block
4. **Transitions:** Simple cuts, no fancy effects needed
5. **Ending:** Include GitHub link and landing page URL on screen
